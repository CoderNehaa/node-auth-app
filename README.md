This is nodeJS authentication project, in which user can create account, sign in to account, reset password when logged in and get new passsword on email if forget password.

Notifications-
User already exists-signup,
Invalid password-signup,
User does not exist-signin,
Incorrect password - signin,

Features - 
Encrypted pasword stored in db.

Libraries used :
bcrypt,
body-parser,
ejs,
express,
express-ejs-layouts,
express-session,
express-validator,
mongodb,
mongoose,
nodemailer,
nodemon
